# Solaire Scripting Editor

Recently I took to visiting some older projects of mine and dug up my previous
attempt at building a code editor in python: Cypher. I've decided to start
again after stumbling across [Aura](https://github.com/rohankishore/Aura-Text).

Its Friday night, right before bed and I have a plane to catch in the morning,
so I won't be going into any details in this post (maybe I'll edit it in the
future). Anyway, here is a few screenshots of my new python code editor:
Solaire.

## Solaire

<img src="https://i.imgur.com/gICHBJy.png">

Solaire is meant to be a standalone code editor that can run python code, or
run inside an art production DCC like Maya, Unreal, Nuke, etc.

### Features

As of writing Solaire has a few fun, albeit simple, features. It has very basic
code analysis, using [Jedi](https://jedi.readthedocs.io/en/latest/), to show
you the structure of your file, it allows for customized syntax highlighting of
Python and Json, various preferences for IDE performance and (very) basic code
formatting options, fun editor features such as code folding for lines ending
in `:`, `(`, `{`, or `[`, and forking stdout and stderr to its own terminal.

<img src="https://i.imgur.com/puUXSvG.png">

Its early days still on this project, but I hope this project matures into
something super special over time.


### PySide6 Widget Library

Developing this application has really helped me develop a PySide6 widget
toolkit library. This library has been fantastic for various projects of mine
and can be found [here](https://github.com/nate-maxwell/PySide6TK)! The
examples folder has various demonstrations for widget uses and implementations.
